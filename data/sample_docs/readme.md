# 示例知识库文档

在此目录下放置你的私有文档，Agent 会自动索引这些文档用于 RAG 检索。

支持的文件格式：
- `.txt` 纯文本文件
- `.md` Markdown 文件
- `.pdf` PDF 文件（需要额外安装 PyPDF2）

## 使用方式

1. 将文档放入 `data/sample_docs/` 目录
2. 启动 Agent 后，在 Web UI 的"知识库管理"标签页上传更多文档
3. 在对话中提问，Agent 会自动从知识库中检索相关信息

## 示例问题

- "这个项目的核心功能是什么？"
- "如何部署这个系统？"
- "支持哪些 AMD GPU 型号？"
